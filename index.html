Projeto Interdisplinar

No Curso Superior Tecnológico em Desenvolvimento de Software Multiplataforma envolve a elaboração de projetos interdisciplinares a cada semestre. Os referidos projetos que o aluno construirá no decorrer do curso subsidiarão o Portfólio Digital que estimula o aprimoramento prático dos estudantes e é valorizado no mercado de trabalho. Faço o cadastro do seu Projeto no botão abaixo ou visualize os temas de semestre anteriores, assim como seu respectivos repositórios no GitHub. ATENÇÃO NA DATA LIMITE DE ENVIO.

Até 27/11

Envie seu projeto até a data limite. NÃO haverá nova data para entregas de projeto durante o semestre, cabendo ao estudante verificar a data e entregar o requerido dentro do prazo.
